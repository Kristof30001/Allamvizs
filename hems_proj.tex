\begin{abstractEN}
This project report studies the optimization of a Home Energy Management System (HEMS) using nature-inspired metaheuristics. We formulate a cost-minimization scheduling problem with practical constraints (e.g., appliance timing, PV generation uncertainty, and battery operation) and compare four approaches: Genetic Algorithm (GA), Particle Swarm Optimization (PSO), Grey Wolf Optimizer (GWO), and a hybrid GA--PSO method. Experiments are evaluated on multiple seasonal scenarios (winter, summer, and cloudy days) using repeated runs to assess robustness. The results highlight that algorithm choice depends on the operating scenario: exploration-focused methods perform better under uncertainty, while constraint-heavy cases benefit from stronger exploitation. Finally, we summarize implementation recommendations and the expected economic and environmental impact.
\end{abstractEN}
