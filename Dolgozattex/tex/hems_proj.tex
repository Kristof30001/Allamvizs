\begin{abstractEN}
This report presents a reproducible benchmark for optimizing Home Energy Management Systems (HEMS) with nature-inspired metaheuristics. The main contribution is the integration of realistic input data on both sides of the energy balance: real day-ahead electricity prices from the ENTSO-E Romanian zone, measured PV generation profiles from a Huawei inverter, and household consumption profiles derived from the UCI Machine Learning Repository smart-meter dataset. To evaluate robustness rather than isolated performance, the study introduces a 2x2 cross-validation matrix that separates synthetic and real PV generation from synthetic and real household load. This design produces 640 algorithm runs across 16 representative days, four data-quality scenarios, five optimizer configurations, and repeated stochastic executions. The results show that input realism significantly changes the optimization landscape: algorithms that perform acceptably under synthetic assumptions can degrade under real PV and UCI-based load variability. Across the full matrix, GA (P=100) provides the most stable cost performance and the strongest robustness in the fully realistic dual-noise scenario, while faster methods such as PSO remain more sensitive to noisy input profiles.
\end{abstractEN}
