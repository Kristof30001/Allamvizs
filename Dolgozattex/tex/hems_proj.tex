\begin{abstractEN}
This report extends the HEMS optimization benchmark by integrating a 2x2 cross-validation matrix to assess algorithm robustness under various uncertainty levels. The optimization core evaluates nature-inspired metaheuristics (GA, PSO, GWO, Hybrid) using a highly realistic setup: real day-ahead electricity prices (ENTSO-E, RO zone), real PV generation profiles from a Huawei inverter, and real-world household consumption data from the UCI Machine Learning Repository. The updated pipeline executed 640 simulation runs, storing outputs in structured formats and generating comprehensive visual analytics. Results indicate that while algorithm ranking is scenario-dependent, the Genetic Algorithm (P=100) exhibits the highest robustness and stability in the fully realistic, dual-noise scenario.
\end{abstractEN}
